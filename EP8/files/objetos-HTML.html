Objetos HTML

Objetos HTML representam documentos na linguagem HTML.

HTML (abreviação para a expressão inglesa HyperText Markup Language , que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção dos padrões HyTime e SGML. [Wikipedia]
